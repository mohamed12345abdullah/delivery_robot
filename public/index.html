How to Use the joy Stick Controller

  • Move the joystick handle to control robot movement in X and Y directions
  • Use the rotation slider to control robot rotation around the Z-axis
  • Click "Connect Robot" to establish connection with the robot
  • Click "Reset Position" to reset the joystick and rotation to default values
  • The robot simulation will respond in real-time to your control inputs
  • Control data shows exact values being sent to the robot
